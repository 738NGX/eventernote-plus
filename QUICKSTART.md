# EventerNote Plus - 快速开始

## 安装步骤

### 1. Chrome/Edge 浏览器

#### Windows 用户
1. 打开浏览器，访问：
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`

2. 开启右上角的"开发者模式"开关

3. 点击"加载已解压的扩展程序"按钮

4. 选择本项目文件夹（eventernote-plus）

5. 完成！访问 https://www.eventernote.com/ 查看效果

#### 验证安装
- 浏览器工具栏出现插件图标
- 访问 EventerNote 网站时，页面会自动应用新UI
- 控制台显示 "EventerNote Plus is loading..." 日志

## 📝 注意事项

### 图标文件
首次使用时，由于缺少图标文件，浏览器可能显示警告。这不影响功能使用。

**解决方法：**
1. 准备 16x16、48x48、128x128 三种尺寸的 PNG 图标
2. 放置到 `icons/` 目录
3. 在扩展管理页面点击"重新加载"按钮

或者暂时移除 manifest.json 中的 icons 配置。

### 使用限制
- 插件仅在 https://www.eventernote.com/* 域名下激活
- 需要网络连接才能访问目标网站
- 某些网站功能可能需要刷新页面才能完全生效

### 调试方法

**查看插件日志：**
1. 在 EventerNote 页面按 F12 打开开发者工具
2. 切换到 Console 标签
3. 查找以 `[ENPlus]` 开头的日志

**调试 Popup：**
1. 右键点击插件图标
2. 选择"检查弹出内容"
3. 在打开的开发者工具中调试

## 🚀 快速定制

### 修改主题色
编辑 `styles/modern-ui.css` 文件中的 `:root` 变量：

```css
:root {
  --enplus-primary: #6366f1;  /* 改成你喜欢的颜色 */
}
```

### 修改搜索行为
编辑 `scripts/search-enhancement.js` 中的配置：

```javascript
// 搜索延迟（毫秒）
ENPlus.debounce((e) => {
  // ...
}, 300);  // 修改这个数字

// 最小搜索长度
if (query.length >= 2) {  // 修改这个数字
  // ...
}
```

## ❓ 常见问题

**Q: 插件没有生效？**
A: 检查是否在 eventernote.com 域名下，刷新页面后重试

**Q: 样式显示不正常？**
A: 清除浏览器缓存，或在扩展管理页面重新加载插件

**Q: 搜索功能没反应？**
A: 检查浏览器控制台是否有错误日志，可能需要等待页面完全加载

**Q: 如何卸载？**
A: 在扩展管理页面点击"移除"按钮即可
