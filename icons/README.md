# EventerNote Plus - 图标资源

由于这是文本环境，无法直接创建PNG图标文件。

## 图标尺寸要求

请准备以下尺寸的图标：
- icon16.png (16x16 像素)
- icon48.png (48x48 像素)  
- icon128.png (128x128 像素)

## 设计建议

1. **主色调**: 使用紫色渐变 (#6366f1 到 #8b5cf6)
2. **图案**: 可以使用笔记本、搜索、星星等元素
3. **风格**: 扁平化、现代简约
4. **背景**: 透明或圆角

## 在线图标生成工具

推荐使用以下工具生成图标：
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
- https://www.canva.com/

## 临时方案

如果暂时没有图标，可以先将 manifest.json 中的 icons 部分注释掉，插件依然可以正常工作。
