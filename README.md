# EventerNote Plus

EventerNote Plus是一款让[EventerNote](https://www.eventernote.com/)更好用的浏览器插件，重新制作了页面并且增加了更多新功能。未来即将上线手机版。本项目遵循MIT协议开源。

## 年度报告

### 如何查看？

每年的12月直到次年1月，首页上都会直接显示当年年度报告的入口栏，直接点击即可进入查看你的年度报告。

你也可以通过访问 `https://www.eventernote.com/annual-report/:year/:username` 来查看年度报告。其中 `:year` 代表你要查看的年份。 `:username` 是目标EventerNote用户的用户名。你可以按需求修改地址查询。

### 如何保存报告？

暂时还不能直接通过页面UI来导出完整报告，需要使用浏览器截图功能。

#### Chrome & Edge 通用

1. F12打开控制台。
2. 键盘 **Ctrl+Shift+P** 启用命令输入。
3. 搜索 **截图**
4. 选择 **截取完整尺寸的屏幕截图（CHrome）捕获全尺寸屏幕截图（Edge）**
5. 截图文件会自动进行下载。

#### Edge快速截图

1. 右键呼出菜单
2. 点击 **截图**
3. 候选项选择 **捕获整页**
4. 此时会弹出一个预览窗口，右上角可以快速选择保存图片或者复制到剪贴板。

## 测试指南

### 打包与分发

运行以下指令：

```sh
npm run release
```

在项目路径下找到一个 `release/` 文件夹。需要注意现在通常Chrome、Edge等浏览器一般会阻止禁用通过 `.crx` 格式文件安装的扩展。因此直接将该文件夹压缩成zip等格式即可进行测试分发。

### 安装

首先解压得到的测试压缩包。

#### Chrome

1. 地址栏输入 `chrome://extensions/` 进入到扩展程序界面。
2. 检查右上角：**开发者模式**已打开。
3. 点击左上角 **加载未打包的扩展程序**，选择你刚刚解压出来的那个文件夹。
4. 应该能在所有扩展程序列表中找到本扩展，确保已经启用。

### Edge

1. 地址栏输入 `edge://extensions/` 进入到扩展程序页面。
2. 检查左侧边栏：**开发人员模式**已打开。
3. 列表标题已安装的扩展右侧：点击**加载解压缩的扩展**，选择你刚刚解压出来的那个文件夹。
4. 应该能在已安装的扩展（来自其他源）中找到本扩展，确保已经启用。
